This is the seminar website for berkeley informal string math seminar. 

It consists of two parts
  - Data: in the data/ folder, in the form of year-semester.MD, like 2026-spring.MD
  - View: in the root folder, index.html, that loads in the data, and displays the talk info.

If you are one of the organizers, and you want to edit the talk schedule, you can click on the quick link
at the bottom of the page 'edit schedule', that will redirect you to the edit page for the corresponding MD file. 
Click commit when you are done. 

When a new semester come, you need to create a new MD file, and also edit the manifest.json file in the data/ folder
to add a new entry.
